# 🗳️ Online Voting System (Python)

A simple command-line based **Online Voting System** built using Python. This project allows registered users to vote for candidates and view voting results while ensuring that each user can vote only once.

\---

## 📁 Project Structure

📦 Online-Voting-System
┣ 📜 main.py
┣ 📜 users.txt
┣ 📜 votes.txt (auto-generated)
┗ 📜 README.md

\---

## 🚀 Features

* User authentication using predefined user IDs
* Vote casting for candidates
* Prevents multiple voting
* Displays vote results
* Persistent vote storage using file handling

\---

## ⚙️ How It Works

1. User enters their User ID
2. System checks if the user exists and hasn’t voted
3. User selects a candidate
4. Vote is saved in votes.txt
5. Results can be viewed anytime
