# 🔐 Password Generator (Python)

A simple Python project to generate strong and random passwords using
letters, numbers, and special characters.

\---

## 📌 Features

* Generate random secure passwords
* Custom password length
* Uses letters (A-Z, a-z), numbers (0-9), and symbols
* Beginner-friendly code

\---

## 🛠️ Technologies Used

* Python 
* Built-in libraries:

  * `random`
  * `string`

## ⚙️ How It Works

* The program combines:

  * Letters (`string.ascii\_letters`)
  * Numbers (`string.digits`)



* It randomly selects characters based on the length entered by the
user.
